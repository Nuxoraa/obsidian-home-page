# Obsidian GitHub Dashboard

Проект представляет собой кастомный дашборд для Obsidian, написанный на dataviewjs. Превращает рабочее пространство в информационный хаб, объединяющий задачи, заметки, погоду и навигацию по файлам.

## Возможности

GitHub тепловая карта. Отображение активности на GitHub за текущий год.

Поиск по файлам. Мгновенный поиск и открытие любых файлов хранилища с подсветкой результата.

Погода. Прогноз на 7 дней для Алматы с почасовым прогнозом при клике. Используется Open-Meteo API.

Календарь с заметками. Встроенный календарь с возможностью добавлять заметки на конкретные даты. Заметки сохраняются локально в браузере Obsidian.

Notepad. Многостраничный блокнот с поддержкой вкладок. Подходит для быстрых идей и черновиков. Сохраняется локально.

Списки задач. Четыре колонки задач с возможностью отмечать выполненное, добавлять и удалять пункты.

Недавние файлы. Быстрый доступ к последним открытым файлам.

Минималистичный дизайн. Темная тема, анимированная сетка и строгая цветовая гамма.

## Установка

Требуется установленный и включенный плагин Dataview в Obsidian. В настройках плагина должно быть разрешено выполнение JavaScript кода.

Создайте новую заметку в хранилище. Скопируйте код из файла dashboard.js. Вставьте код в заметку, обернув в блок кода dataviewjs.

## Настройка

Для изменения пользователя GitHub отредактируйте строку const USERNAME = "nuxoraa" в начале кода.

Для изменения города для погоды измените параметры latitude и longitude в fetch запросе.

Массив CARDS определяет заголовки колонок и начальные задачи.

## Использование

GitHub карта. Наведение на квадрат показывает количество контрибьюшенов и дату.

Поиск. Ввод названия файла выводит результаты под строкой поиска. Клик по результату открывает файл.

Погода. Основной вид показывает недельный прогноз. Клик по дню разворачивает подробную панель справа. Повторный клик сворачивает панель.

Календарь. Навигация стрелками или кнопкой Today. Клик по числу открывает окно для ввода заметки. Сохранение отмечает дату точкой.

Notepad. Переключение между вкладками кликом. Двойной клик по названию вкладки для переименования. Кнопка плюс создает новую вкладку. Кнопка закрытия появляется при наличии более одной вкладки. Текст сохраняется автоматически.

Списки задач. Клик по квадрату слева отмечает задачу выполненной. Поле внизу колонки добавляет новую задачу. Крестик справа удаляет задачу.

## Хранение данных

Все пользовательские данные сохраняются в localStorage браузера Obsidian. Данные привязаны к конкретному компьютеру и не синхронизируются через Obsidian Sync, Git или другие сервисы автоматически. При переустановке Obsidian или очистке кэша данные могут быть потеряны.
